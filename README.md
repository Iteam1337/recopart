# recopart

##Pre-requisites

```bash
npm install -g gulp
npm install -g cordova
npm install -g bower
```

##Setup

Clone the repo
```bash
git clone https://github.com/Iteam1337/recopart.git
```
Install dependencies
```bash
npm install
bower install
```
Build the project
```bash
gulp build
```
Run the app
```bash
cordova run ios
cordova run browser
```
